# Overview

`OpenBench` is a lightweight evaluation harness that tracks 40+ trustworthy AI tasks, from adversarial robustness to calibration drift.

## Features

- YAML-first task definitions and reproducible docker recipes
- Live leaderboard widgets embeddable into lab pages
- Automated badge generation for open-source repos

## Status

Actively used by more than 35 research groups; incubated under the TrustML community license.
